Downloads
=========

Readily-built VM images
-----------------------

Download readily-built VM images `here <http://tinyurl.com/OpenRadarVM>`_.

Make sure you select the correct image for your system (32-bit or 64-bit built).

These images can be instantly run by using `VirtualBox <http://www.virtualbox.org>`_.


Download source
---------------

Source distributions can be downloaded `here <https://github.com/openradar/oss_weather_radar_vm/archive/master.zip>`_. 

See :doc:`here <build>` how to build the VM from source by using `Vagrant <http://www.vagrantup.com>`_
