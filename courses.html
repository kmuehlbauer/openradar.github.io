The course materials available so far are mainly based on the Open Source Radar Short Course which was carried out at the ERAD 2014 for the very first time.

The courses are based on the IPython notebook.

Once you have logged into the VM, run the command

$ ./start_notebook.sh.

Then open a browser window on your host machine and navigate to http://127.0.0.0:8888/tree. From here, you can browse through the available IPython notebooks.

You can also access the course repositories directly:

  • Py-ART course
  • wradlib course
  • BALTRAD course
  • BALTRAD 2 wradlib interoperability demo
  • Py-ART 2 BALTRAD interoperability demo
  • Radx course (as yet very brief)
